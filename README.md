# Amateur Experimental Rocketry 🚀

This repository documents an undergraduate experimental rocketry project focused on the **design, fabrication, and testing of a low-powered solid-propellant model rocket**, developed through a structured, multi-phase engineering approach.

The core goal of the project is to move beyond simulation-only hobby rocketry and explore **hands-on propulsion, stability, avionics, and control systems**, while keeping the design accessible, modular, and reproducible.

## Project Overview

The rocket is developed incrementally across four phases:

- **Phase I – Baseline Rocket Validation**  
  Design and validation of a stable airframe and solid rocket motor using KNSu-family propellants, including static thrust testing and aerodynamic stability analysis.

- **Phase II – Recovery System Integration**  
  Design and testing of a parachute-based recovery system using ejection charges to ensure safe and reusable flight operations.

- **Phase III – Avionics & Telemetry**  
  Integration of onboard avionics for GPS-based telemetry and a ground station for real-time flight data monitoring.

- **Phase IV – Thrust Vector Control (TVC)**  
  Implementation of a servo-driven nozzle gimbal system to actively control rocket attitude during powered ascent.

## What This Repo Contains

- Design notes and documentation
- LaTeX project reports
- Simulation files and calculations
- Propulsion test data
- Avionics and control system experiments
- Iterative design decisions and lessons learned

## Why This Project Exists

This project is an attempt to **understand rocketry from first principles** — combustion, aerodynamics, structures, electronics, and control — rather than treating rockets as black-box kits.  
It also serves as a small step toward exploring **reusable booster concepts** at a model scale, using simple materials and student-accessible tooling.

> Learn by building. Validate by testing. Improve by iteration.
